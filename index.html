Mi Portafolio

01

Gestor de Proyectos

Thesis Project Manager es una aplicación web full-stack para la gestión integral de proyectos. Su arquitectura desacoplada consiste en un backend con Node.js que expone una API RESTful y utiliza Prisma como ORM para interactuar con una base de datos relacional. El frontend es una SPA desarrollada con Next.js y TypeScript, estilizada con Tailwind CSS para una interfaz moderna y responsiva. La aplicación ofrece funcionalidades clave como autenticación de usuarios, múltiples vistas de proyectos (tablero, lista, cronograma), gestión de tareas por prioridad y búsqueda avanzada, demostrando una sólida implementación de un sistema de gestión de proyectos completo y eficiente.

Next.js Node.js Express Prisma PostgreSQL
Repositorio de GitHub

02

Exquisite Coffee

Exquisite Coffee es una aplicación web full-stack que optimiza la toma y gestión de pedidos en una cafetería, ofreciendo una experiencia fluida al cliente. Utiliza un backend en Laravel con más de 15 endpoints RESTful y una base de datos relacional de 8 tablas, junto con un frontend en React (SPA) estilizado con Tailwind CSS. Implementa autenticación con roles, gestión de estado global con React Context, y SWR para sincronización eficiente en tiempo real. La integración entre frontend y backend demuestra habilidades sólidas en desarrollo web desacoplado, orientado a la eficiencia operativa.

Laravel React Tailwind SWR PHP JavaScript
